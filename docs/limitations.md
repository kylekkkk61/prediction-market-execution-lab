# Limitations

This project is designed as a public research and portfolio project. The following limitations should be explicit throughout the repository.

## Research Limitations

- Short-horizon prediction markets can be noisy and path-dependent.
- Market prices may reflect liquidity constraints as much as information.
- Backtests do not guarantee live execution outcomes.
- Simulated fills may differ from real order-book behavior.
- Sample data may be simplified or anonymized for public release.
- Model outputs should not be interpreted as recommendations.

## Public Repository Boundaries

The public version should not expose:

- private keys,
- wallet addresses tied to private activity,
- signer or relayer configuration,
- live deployment runbooks,
- private ledgers,
- complete production execution flows,
- strategy-sensitive thresholds.

## Intended Use

This repository is intended for research, education, and portfolio demonstration. It is not financial advice, trading advice, or a production execution system.
